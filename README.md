# Usage

**Option 1: clone or download this repository**

**Option 2: use [static-site-generator-cli](https://github.com/MudOnTire/static-site-generator-cli/tree/master) tool**

## 📦 Install dependencies

```sh
yarn
```

## 🔨 Dev

```sh
yarn start
```

## 🌍 Build

```sh
yarn build
```
